# Examples

Run any script with `python examples/<script>.py`.

- `robust_supply_planning.py`: End-to-end training + conformal calibration + calibration curve + robust planning.
- `robust_shortest_path_metrla.py`: Conformalized DCRNN forecasts + robust shortest path on METR-LA.
- `robust_bike_newsvendor.py`: Conformal calibration on UCI Bike Sharing data + robust newsvendor decisions vs nominal.
- `robust_portfolio.py`: Conformalized predictor + robust portfolio over synthetic asset returns (Wine Quality features).

See `examples/README.md` for mathematical formulations.
